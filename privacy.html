Privacy Policy

Last updated: April 2026

Antares does not collect, store, or transmit any personal data. Period.

What We Collect

Nothing. Antares runs entirely in your browser. No accounts, no sign-ups, no analytics, no cookies, no tracking pixels.

How It Works

When you visit a page containing a Solana token address, the extension queries public APIs (DexScreener, RugCheck, GoPlus, Helius, Solscan) to fetch token data. These requests are made directly from your browser. Antares does not proxy, log, or intercept any of this data.

Data Storage

All data is processed locally and discarded after the scan result is displayed. Nothing is saved to disk, cloud, or any server.

Third-Party Services

Antares queries the following public APIs for token data:

  • DexScreener API
  • RugCheck API
  • GoPlus Security API
  • Helius RPC
  • Solscan API

These services have their own privacy policies. Antares does not send any personal information to them — only the token address being scanned.

Permissions

The Chrome extension requires minimal permissions to detect Solana addresses on web pages and display scan results. No access to passwords, forms, or browsing history.
