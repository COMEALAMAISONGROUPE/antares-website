Analysis Layers
L1

Liquidity Analysis

DexScreener

Evaluates total liquidity depth across all active pairs. Near-zero liquidity is a Safe Gate trigger. Thresholds scale with market cap to avoid false positives on micro-caps.

L2

Contract Security

GoPlus

Contract security analysis: proxy contracts, blacklist functions, transfer restrictions, hidden fees, and sell restrictions. Any critical finding triggers Safe Gate instantly.

L3

Authority Check

Helius + RugCheck

Verifies mint authority and freeze authority directly on-chain. Active mint means infinite token printing. Active freeze means your wallet can be locked. Both are Safe Gate triggers.

L4

LP Status

RugCheck

Checks whether LP tokens are burned (permanent), locked (time-limited), or still held by the deployer (rug risk). Burned LP is the safest outcome. Unlocked LP with concentrated ownership deducts heavily.

L5

Holder Distribution

Solscan

Analyses top wallet concentration and holder count. Top 10 wallets holding >50% supply triggers major deductions. Combined with token age, this catches pre-rug setups.

L6

Chart Intelligence

GeckoTerminal

OHLCV candle analysis across 5m, 1h, 6h, 24h windows. Detects parabolic pumps, blow-off tops, and wash trading. If no candle data is available, L6 is skipped gracefully.

SAFE GATE

The Safe Gate is a hardcoded override that fires before the numeric score is even calculated. If any of these conditions are met, the token is blocked immediately. No score, no appeal, no override.
