Scoring Engine

7 LAYERS.
1000 POINTS.

Seven independent analysis layers each contribute weighted deductions to a 0–1000 composite score. The score is the geometric mean of layer trusts — one bad source can’t hide; one false-positive can’t flip a clean token to RUG. The Safe Gate overrides on critical hard flags (LP open, honeypot, mint/freeze active, deceptive name).

Score Breakdown — 5 Dimensions

5-Axis Pentagon

The 1000-point score is decomposed across five orthogonal dimensions. The radar shows precisely where a token passes and where it fails — instead of a single mysterious gauge.

  • LP Security — burn / lock status, extractable USD
  • Holder Distribution — top 10 concentration, single-wallet risk
  • Trading Authenticity — vol/liq ratio, wash signals
  • Token Maturity — on-chain age, holder growth curve
  • Source Consensus — how many upstreams agree
Detection Layers
L1

Liquidity & Market

DexScreener

Real-time price, volume, liquidity depth, transaction counts across 5m, 1h, 6h, 24h. Near-zero liquidity is a Safe Gate trigger. Volume/liquidity ratio feeds wash trading detection.

L2

Token Audit

RugCheck

LP burn/lock status, metadata mutability, mint & freeze authority, bundler detection, sniper activity. Solana-native token-audit signal.

L3

Contract Security

GoPlus

Honeypot simulation, hidden fees, transfer restrictions, blacklist functions, sell tax / buy tax, anti-whale rules. Honeypot detection is an immediate Safe Gate trigger.

L4

On-Chain Truth

Helius RPC

Direct Solana RPC: top 20 holders (filtered for LP/foundation wallets), token supply, mint authority, creator wallet. The ground-truth layer that bypasses any aggregator.

L5

Holder Distribution

Solscan

Holder count, on-chain token age, recent transfer history (powers the live Holder Activity tab), 24h trades / unique traders. Catches pre-rug concentration setups.

L6

Chart Intelligence

In-house Chart Engine

OHLCV candle analysis across 5m / 1h / 6h / 24h. Detects parabolic pumps, blow-off tops, staircase manipulation, wick traps, dead-cat bounces. Skipped gracefully if candle data is missing.

L7

Cross-Validation

Internal

Detects disagreements between layers (e.g. LP burn claimed by RugCheck but LP program still holds tokens per Helius). Disagreement triggers a 0.7–0.9× penalty so a single suspicious source can’t mask a hidden risk.

SAFE GATE

Hard verdict overrides — even a high numeric score can’t unlock a token if any of these is true. They reflect risks that can never be soft-unlocked by maturity or holder count.

✗ LP not burned / locked ✗ Honeypot — cannot sell ✗ Mint authority active ✗ Freeze authority active ✗ Deceptive name (impersonation) ✗ Bundle / sniper detected ✗ Liquidity < $500 ✗ < 15 holders
What Gets Surfaced

Time-to-Rug Ring

For RUG & DANGER verdicts, the hero shows a circular ring with the median time-to-rug for comparable launches (e.g. 4h 12m). Heuristic profile match against historical clusters — indicative, not a forecast.

Critical Actors

Three cards: Dev (with prior-token reputation), Insider (largest non-creator holder), Cluster (sibling-wallet coordination score detected via insider-graph union-find).

Holder Activity (live)

Last-60-min flow per top holder, classified Holding / Selling / Splitting / Reducing / Buying / Static. Net-flow footer (-3.2% pre-dump signature) catches dev/whale dumps in real time.

Verdict Timeline

Per-token verdict history persisted server-side (Redis ZSET). Watch a token bounce CAUTION → DANGER → RUG across recent scans, with the event that triggered each shift.

Outcome Histogram

36-bucket distribution of comparable launches over time-to-rug, current token marked with a YOU bar. Three most-similar past launches with their actual losses.

AI Verdict (Gemini)

2–4 sentence plain-English synthesis — references actual data (dev wallet history, holder concentration, last-hour flow). Local fallback if the LLM is unreachable.
